# Заготовка для коллекции статей

## Для чего подходит
* блог
* новостной портал
* коллекция статей

## Как запустить версию для разработки
```
yarn install && yarn dev
```

## Что сделано
* эмитация авторизации (с сохранением в localStorage)
  * user: Admin
  * password: 12345
* защищеные роуты
* статьи (эмитация без сервисов)
